# 通用错误码 #

| code | 说明 |
| ---- | ---- |
| 2000 | 系统错误，参数错误等 |

# 公会接口 #

# 检测公会名 #
> 
>##   /guild/checkGuildName ##
> #### 参数
>
| 参数        | 类型    |  说明     |
| --------   | -----   | ----     |
| guild_name | string  |   公会名   |

> #### 业务错误码 ####
>
| code | 说明  |
| ---- | ---- |
| 20103 | 公会名为空 |
| 20102 | 公会名已存在 |
| 20101 | 公会名过长 |
| 0    |  公会名可用 |

# 创建公会 #
## /guild/create ##
>####参数
>     
|参数      |类型         |说明 |
| ---- | ---- | ---- |
|camp      |string      |阵营 1或2 |
|guild_name|string      |公会名 |
|guild_describe|string      |公会简介,可选|
>#### 业务错误码
>
|code|说明 |
| ---- | ---- |
|20103|公会名为空|
|20102|公会名已存在|
|20101|公会名过长|
|0    |成功|

# 申请入会 #
## /guild/apply ##
>#### 参数 ####
>
| 参数     | 类型      | 说明  |
| ----     | ----     | ---- |
| guild_id | string   |公会id |

>#### 业务错误码 ####

| code | 说明 |
| ---- | ---- |
| 20206 | 退会冷却 |
| 20202 | 已经申请 |
| 20203 | 公会已关闭申请 |
| 0    | 成功 |
>#### 返回值
>如果退会冷却， result返回剩余剩余时间，秒

# 入会 #
## /guild/join ##
>#### 参数 ####
>
| 参数     | 类型      | 说明  |
| ----     | ----     | ---- |
| guild_id | string   |公会id |
| join_user_id | array | 字符串，批量入会用逗号分开 要入会人的id|

>#### 业务错误码
>
| code | 说明 |
| ---- | ---- |
| 20204 | 公会人数已达上限 |
| 20201 | 已经加入公会 |
| 20203 | 公会已关闭申请 |
| 20208 | 没有权限 |
| 20209 | 没有此条申请 |
| 0    | 成功 |
